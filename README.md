# AI Development Environment Setup

## Overview

This repository documents the setup process I followed for this portfolio project. It includes the tools I installed, the steps I completed, and a few challenges I encountered along the way.

## Tools Installed

* Cursor IDE (Version 3.9.16)
* GitHub
* Claude Code for VS Code
* Codex – OpenAI's coding agent

## Steps Completed

1. Installed Cursor IDE.
2. Created a GitHub account.
3. Installed the Claude Code and Codex extensions.
4. Created a public GitHub repository.
5. Opened the repository in Cursor.
6. Created this README to document the setup process.

## Issues Encountered

At first, I couldn't find the requested extensions because the current version of Cursor has a different interface from the tutorials I found. After exploring the Marketplace and checking the available options, I located the official Claude Code and Codex extensions and completed the setup.

## What I Learned

This project reminded me that software changes quickly. When something doesn't match the instructions, it's important to check the latest documentation, adapt to the current version, and keep moving forward.
